\# 🚀 MemeConnect

> "WASA WASA WASA WASA... MEMECONNEEEEEEECT!"



MemeConnect is a high-performance, cross-server meme relay bot. It connects 

independent Discord communities into a single, global meme-sharing network 

using a centralized moderation and "Quarantine" architecture.



\## 🛠️ Key Features

\- \*\*Global Relay:\*\* Post once, broadcast to all connected servers.

\- \*\*AI-Powered Filtering:\*\* Uses pHash (Perceptual Hashing) to detect and block 

&#x20; banned memes instantly, even if they are slightly edited.

\- \*\*The Quarantine:\*\* Suspicious content is hidden and sent to a private Staff 

&#x20; Server for manual approval/rejection via Discord Buttons.

\- \*\*Global Ban System:\*\* One strike on the network can lead to a global ban 

&#x20; across all servers using the bot.

\- \*\*Staff Badges:\*\* Official moderators get a unique 🛡️ badge in the relay.



\## 🏗️ Architecture

\- \*\*Language:\*\* Python (discord.py)

\- \*\*Database:\*\* SQLite (aiosqlite) for lightweight, fast data management.

\- \*\*Image Processing:\*\* Pillow + ImageHash for digital fingerprinting.



\## 🚀 Setup

1\. Clone the repository.

2\. Install dependencies: `pip install -r requirements.txt`.

3\. Create a `.env` file and add your `BOT\_TOKEN`.

4\. Run `python main.py` to start the revolution.
